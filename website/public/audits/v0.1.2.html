The biggest quality-of-engineering release so far. 34 automated tests appear from nothing, database queries move to compile-time verification, migrations replace inline SQL, and contributor infrastructure (CHANGELOG, CONTRIBUTING.md, Dependabot) lands all at once. The biggest remaining gaps are logging/observability, a PR CI workflow, macOS/Linux builds, and the misleading games table.
Standout additions in v0.1.2
Compile-time SQL safety via sqlx macros NEW
All database queries migrated from runtime strings to query! / query_as! macros. SQL errors now surface at compile time, not in production. Combined with the cargo sqlx prepare offline cache, CI builds no longer require a live database connection. This is a sophisticated production-engineering choice.
34 tests from zero NEW
12 Rust unit tests (ports, recipes, error serialization), 9 SQLite DB integration tests running against an in-memory database, and 13 TypeScript/Svelte API tests via Vitest. Going from zero to a meaningful test suite in a single release is a strong signal of discipline.
