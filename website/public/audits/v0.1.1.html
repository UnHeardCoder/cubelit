What was fixed in v0.1.1
✓

Switched to the correct TAURI_SIGNING_PRIVATE_KEY env var, strips carriage returns, writes key to temp file to preserve newlines. Auto-update now actually delivers updates to users.
✓
Windows onboarding completely rebuilt
WSL2 detection, one-click enable via UAC prompt, separate screens for each step (WSL2 → Docker install → Docker running). Users are guided rather than dropped into a broken state.
✓
Microsoft Store link fixed
Was pointing to the wrong app. Now goes to Docker Desktop directly.
✓
Windows null pointer bug fixed
Null pointer type for ShellExecuteW HWND parameter corrected — was causing crashes on certain Windows configurations.
